# vibe-local-windows
Windows port of ochyai/vibe-local - Free AI coding environment: Ollama + Claude Code CLI on Windows

## vibe-pet 🐣

A Codex-style terminal pet that grows while you code. It hatches from an egg, gains XP every time Claude Code works, and can live in your Claude Code status line.

```sh
node pet/pet.js          # live view (f = feed, p = play, q = quit)
node pet/pet.js status   # quick status card
```

See [pet/README.md](pet/README.md) for details and Claude Code integration (日本語ドキュメントあり).
